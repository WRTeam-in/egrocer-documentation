# Self Pickup Settings

The admin can enable or disable the self pickup feature globally for all sellers. When enabled, individual sellers can then configure their own self pickup settings.

![Admin Self Pickup Settings](/img/admin-panel/selfpickup.png)

## Overview

Self pickup allows customers to collect their orders directly from your physical store location. This feature is particularly useful for local businesses and can help reduce delivery costs while providing customers with more flexibility.

### Seller Individual Settings

Once the admin enables the self pickup feature, individual sellers can configure their own self pickup options including:

- **Store Location**: Set pickup address and location details
- **Store Timings**: Set timings for pickup

![Admin Self Pickup Settings](/img/admin-panel/seller-selfpickup.png)


## Configuration Steps

### For Admin (Global Settings)

1. **Access Admin Panel Settings**

   - Log in to your admin panel
   - Navigate to Settings â†’ Store Settings
   - Look for "Seller settings" section

2. **Enable Global Self Pickup**

   - Toggle the "Enable Self Pickup" option
   - This enables the feature for all sellers
   - Sellers can now configure individual settings

### For Sellers (Individual Settings)

1. **Access Seller Dashboard**

   - Log in to seller panel
   - Navigate to My profile
   - Look for "Self Pickup Mode" section (only visible if admin enabled it)

2. **Configure Seller Self Pickup**

   - Set store location
   - Enter store address
   - Set operating hours for pickup

## Customer Experience

### Order Placement

1. Customer selects "Self Pickup" during checkout
2. Chooses preferred time slot
3. Receives confirmation with pickup details
4. Gets notification when order is ready

### Pickup Process

1. Customer arrives at store
2. Shows order confirmation
3. Staff verifies order details
4. Customer collects order
